\begin{abstract}
This project presents a heuristic methodology for solving the School Bus Routing Problem using real road-network data from Medellín, Colombia.
The proposed approach integrates graphs extracted from OpenStreetMap, capacitated clustering using k-medoids with minimum-cost flow assignment, and route optimization through Google OR-Tools.
The system models a single school as the main depot and assigns student stops to buses under capacity constraints.
Distances are computed over the actual road network using shortest paths, after which students are grouped while respecting vehicle capacities, and finally a closed route is optimized for each bus using a Traveling Salesman Problem (TSP)-style formulation.
The methodology generates feasible and efficient routes while minimizing the total fleet distance in a realistic urban transportation environment.
\end{abstract}
